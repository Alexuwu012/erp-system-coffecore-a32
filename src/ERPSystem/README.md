# ERP System - Backend

Esta carpeta contiene el código fuente del sistema ERP.

El backend del sistema será desarrollado en C# utilizando la plataforma .NET y será responsable de gestionar la lógica de negocio del sistema.

## Funcionalidades del sistema

El sistema ERP incluirá los siguientes módulos:

- Administración (EP-ADM)
- Gestión de Clientes (EP-CRM)
- Inventario (EP-INV)
- Proveedores (EP-PROV)
- Ventas (EP-VEN)
- Recursos Humanos (EP-RRHH)
- Finanzas (EP-FIN)

## Arquitectura

El sistema seguirá una arquitectura cliente-servidor donde:

- El cliente accede mediante una interfaz web.
- El servidor de aplicación ejecuta la lógica del sistema.
- La base de datos almacena la información empresarial.

## Estado actual

Actualmente el proyecto se encuentra en la fase de desarrollo de la base de datos (Sprint 1).  
En los siguientes sprints se implementará la API y la lógica de negocio del sistema.
