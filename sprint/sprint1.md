# Sprint 1

## Sprint Goal
Desarrollar la estructura inicial de la base de datos del sistema ERP y definir la arquitectura base documentada en arc42.

## Resultados

Durante este sprint se implementaron:

- Modelo relacional de la base de datos
- Tablas de los módulos:
  - Administración
  - Clientes
  - Proveedores
  - Inventario
  - Compras
  - Ventas
  - Recursos Humanos
  - Finanzas

También se implementaron triggers para:

- Actualizar inventario automáticamente
- Registrar movimientos financieros
